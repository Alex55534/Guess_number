\# 🎲 Игра «Угадай число»


Простая консольная игра на Python, в которой компьютер загадывает число от 1 до 10, а игрок должен угадать его за 3 попытки.


\## 🚀 Как запустить

1\. Убедитесь, что у вас установлен Python 3.6 или выше.

2\. Склонируйте репозиторий или скачайте файл с кодом.

3\. Запустите игру командой:



```bash

python main.py

🕹️ Правила игры

Компьютер случайным образом выбирает число от 1 до 10.

У вас есть 3 попытки, чтобы угадать это число.

После каждой неудачной попытки количество оставшихся попыток уменьшается.

Если вы угадываете число, программа поздравляет вас случайной фразой.

Если попытки заканчиваются, игра завершается поражением.

После окончания раунда можно сыграть ещё раз.

📦 Зависимости

Проект не требует сторонних библиотек. Используется только встроенный модуль random.
